# 🎥 Live Object Detection & Tracing App

A real-time object detection and tracking web application built with Streamlit, WebRTC, and YOLOv8.

This app uses your webcam to detect and track objects live directly in the browser.

---

## 🚀 Features

- 📷 Real-time webcam streaming
- 🤖 YOLOv8 object detection
- 🎯 Object tracking with IDs
- ⚡ Fast performance using YOLOv8n
- 🌐 Browser-based interface using Streamlit

---

## 🛠️ Technologies Used

- Python
- Streamlit
- streamlit-webrtc
- OpenCV
- Ultralytics YOLOv8
- AV
